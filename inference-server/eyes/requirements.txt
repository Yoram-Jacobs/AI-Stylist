# DressApp Eyes v4 — Transformers + PEFT runtime.
#
# Replaces the v3 llama-server proxy. We load `google/gemma-4-E2B-it`
# directly via transformers, quantize to int4 on CPU via optimum-quanto,
# and apply our Eyes v4 LoRA adapter from /adapter via peft.
#
# Pins explained
# ==============
#   torch (CPU)       — the +cpu wheel is installed via --extra-index-url
#                       in the Dockerfile. We pin a 2.4.x line so the
#                       bf16-on-CPU intrinsics are present (added in 2.1)
#                       AND the quanto kernels are stable.
#   transformers      — 4.57+ exposes ``AutoModelForMultimodalLM`` (the
#                       class Google's audio docs use for Gemma-4) and
#                       integrates ``QuantoConfig``. main.py also probes
#                       ``AutoModelForImageTextToText`` for older builds.
#   peft              — 0.13+ added Gemma-4 LoRA target-module heuristics
#                       and fixed merge_and_unload for clipped Linears.
#   optimum-quanto    — int4/int8 weight-only quant that actually works
#                       on x86 CPU. bitsandbytes' CPU path is still
#                       experimental as of mid-2026; quanto is the safe
#                       default.
#   accelerate        — required by transformers' device_map="cpu"
#                       and the from_pretrained meta-init pathway.
#   librosa+soundfile — audio decode + 16 kHz mono float32 resample for
#                       /transcribe (Gemma-4 audio tower input shape).
#   python-multipart  — FastAPI's File()/UploadFile() requires this; it's
#                       NOT a default transitive dep of fastapi[standard].
#   huggingface_hub   — gated download of google/gemma-4-E2B-it; the
#                       transformers `from_pretrained(token=...)` call
#                       routes through this.
fastapi==0.115.6
uvicorn[standard]==0.32.1
python-multipart==0.0.20
pydantic==2.10.4
httpx==0.28.1
huggingface_hub>=0.34.0,<1.0

# the version that landed AutoModelForMultimodalLM + Gemma-4 support.
